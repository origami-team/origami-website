Handbuch

Navigationsaufgabe

OriGami kann auf unterschiedlichen mobilen Endgeräten verwendet werden. Dieses Handbuch ist speziell für die Nutzung und Anwendung mit dem IPad konzipiert.

Wiederkehrende Bedienelemente

In der App finden sich einige wiederkehrende Symbole als Bedienelemente, welche an dieser Stelle kurz beschrieben werden:

Durch das Klicken dieses Pfeilsymbols gelangst Du auf die vorherige Seite.
Durch das Klicken dieses Kreuzsymbols wird das Fenster geschlossen.
Durch das Klicken dieses Informationssymbols erhälst Du weiterführende Informationen zu dem jeweiligen Bereich.
Das Fragezeichensymbol kennzeichnet weiterführende Informationen.
Durch das Klicken dieses Diskettensymbols werden Deine Aufgaben gespeichert.

Hauptmenü

Nach dem Öffnen der App erscheint das Hauptmenü von OriGami. In der aktuellen Version der App stehen fünf Bereichen zur Verfügung. Diese sind im Folgenden abgebildet.

Es gibt verschiedene Aufgabentypen. Der Fokus liegt nachfolgend auf der Erstellung von Navigationsaufgaben, welche schrittweise angeleitet werden.

Kostenlose Registrierung und Login

Um eine Navigationsaufgabe erstellen zu können, ist eine kostenlose Registrierung erforderlich. Hierfür kannst du einen beliebigen Nutzername und ein beliebiges Passwort auswählen.

Nach erfolgreicher Registrierung und erneutem Login erscheint das Hauptmenü erneut. Nun stehen Dir sechs Bereiche zur Verfügung. Wähle den zweiten Bereich Entwickeln aus.

Navigationsaufgabe erstellen

Schritt 1: Spiel erstellen

Es erscheinen folgende Auswahlmöglichkeiten. Klicke auf den Bereich Erstellen um ein eigenes Spiel zu gestalten.

Es stehen Dir drei verschiedene Aufgabentypen zur Verfügung. Klicke zur Erstellung von Navigationsaufgaben auf das Pfeilsymbol.

Schritt 2: Navigationsaufgabe

Es erscheint das Fenster Navigationsaufgabe.

Im Bereich der Navigationsaufgaben stehen Dir die folgenden vier verschiedenen Aufgabentypen zur Verfügung: Navigation zur Zielfahne, Navigation mit Richtungspfeil, Navigation mit Textanweisung und Navigation mit Foto. Im Folgenden wird die Navigation zur Zielfahne exemplarisch erläutert.

Schritt 3: Navigationzur Zielfahne

Wenn Du die Navigation zur Zielfahne auswählst, stehen Dir die folgenden Auswahlmöglichkeiten zur Verfügung: Aufgabenstellung, Audiobotschaft, Zielort und weitere Einstellungen.

Schritt 3a: Aufgabenstellung

Falls Du keine Änderung in der Aufgabenstellung vornimmst, verwendet OriGami automatisch den Satz „Gehe zur Fahne“. Optional kannst Du eine abweichende Aufgabenstellung formulieren.

Schritt 3b: Audiobotschaft

Auch eine Audiobotschaft mit weiteren Anweisungen oder zur auditiven Wiedergabe der Aufgabenstellung kannst Du optional ergänzen.

Schritt 3c: Zielort

Klicke auf einen gewünschten Ort auf der Karte, an den die Kinder navigiert werden sollen.
Eine Aufgabe kann bereits durch das Setzen eines Zielorts auf der Karte vollständig erstellt werden.

Schritt 3d: Weitere Einstellungen

Auch weitere Einstellungen, wie zum Beispiel Feedback über korrekt gelöste Aufgaben, können von Dir optional ergänzt werden.

Schritt 3e: Aufgabe speichern

Klicke auf die Speicherdiskette in der rechten oberen Ecke um Dein Spiel zu speichern.

Schritt 4: Bearbeiten der eigenen Aufgaben

Nach dem Speichern Deiner Aufgabe findst Du diese in der Übersicht. Hier könntest Du dieses weiterbearbeiten.

Schritt 5: Erstellen weiterer Aufgaben

Wenn Du weitere Aufgabentypen hinzufügen möchten, starte wieder ab Schritt 1: Spiel erstellen.

Schritt 6: Spiel speichern

Klicke auf das Diskettensymbol oben rechts.

Es erscheint das folgende Bild. Hier kannst Du Deinen Spielnamen und Spielort eingeben. Klicke anschließend auf Spiel speichern>.

Es erscheint das folgende Bild. Dein Spiel wurde gespeichert. Klicke auf Fertig, um wieder in das Hauptmenü zu gelangen.

Fertiges Spiel spielen

Es erscheinen folgende Auswahlmöglichkeiten. Klicke auf den Bereich Spielen, um ein eigenes Spiel zu gestalten.

Es erscheint eine Liste mit allen in der App OriGami erstellten Spielen. Klicke auf Dein erstelltes Spiel Münster-Schloss-neuer Nutzer.

Klicke auf Start, um Dein Spiel zu starten.

Gib ein, wer spielt und klicke auf Fertig.

Das Spiel startet. Unten siehst Du die Aufgabe, die Aufgabenstellung und den Aufgabentyp anhand des Symbols. Falls Du eine Audiobotschaft hinterlegt hast, erscheint diese hier ebenfalls.

Exemplarisch hast Du jetzt gesehen, wie Du eine Navigation zur Zielfahne erstellen und spielen kannst. Gerne kannst Du weitere Aufgaben in Deinem Spiel ergänzen und das Spiel individuell an Deine Lerngruppe anpassen.
